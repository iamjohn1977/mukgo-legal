# MukGo Marketing Package

The complete promotional toolkit for MukGo — one message system across investors, users,
and press. All copy is **English-first** (US market); non-English versions exist only where
the reader is non-English (app store localization, Korean community/press).

**Master promise:** *Stop with confidence.*
**Category:** a decision engine for drivers · **North Star:** *"Would you stop here again?"*

---

## Start here

👉 **`brand-assets.md` is the source of truth.** Before writing or changing any copy
anywhere, check it. It holds the brand statement, tagline hierarchy, North Star, moat line,
voice rules, and the "don't do this" rules. Everything else follows it.

## How to run it (operating rules)

1. **`brand-assets.md` is the constitution.** If new copy conflicts with it, the copy is
   wrong — or the brand doc needs a deliberate update first.
2. **Edit the source, then redeploy.** Don't hand-edit copy on the platform. Change the `.md`
   file here, then re-paste. That keeps one version of the truth.
3. **English is the base.** Only localize when the audience isn't English-speaking.
4. **The through-line, everywhere:** category = *decision engine for drivers*; promise =
   *Stop with confidence*; moat = *"Google remembers places. MukGo remembers you."*
5. **Never:** name another company's brand in public materials · attack Google · say
   "we're changing the world."

## Suggested rollout order (maps to the 90-day plan)

1. **Weeks 1–3** — Update the App Store listing + homepage. Publish the founder story.
2. **Weeks 4–7** — Community seeding (Korean/Asian driver communities + road-trip subreddits).
3. **Weeks 6–9** — Product Hunt launch + PR (the "app that rates restrooms" angle). Turn on subs.
4. **Weeks 8–12** — Video series + referral. Investor outreach once traction moves.
